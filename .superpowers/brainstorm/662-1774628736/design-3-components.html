CSS 改动范围摘要

  • css/variable.css — 重建 token 系统（颜色 + 形态 + 间距），新增 6 套主题，旧变量名保留别名
  • tab/src/css/search.css — 搜索框高度、圆角、阴影、按钮样式
  • tab/src/css/link.css — 集合头部/正文统一圆角、链接卡片圆角升级、hover 动效
  • tab/src/css/home.css — 侧边栏宽度、图标尺寸、active 态光晕阴影
  • tab/src/css/dropdown.css — 圆角、padding、item hover 样式、分隔线
  • tab/src/css/tabs.css — toolbar 区域间距和按钮样式微调
  • tab/src/js/theme-switcher.js — THEMES 数组更新为 6 套新主题名
  • tab/index.html — theme-option data-theme 属性和显示名称更新
